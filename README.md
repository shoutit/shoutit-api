### OS Requirements
- tesseract-ocr


### Env variables

- `SHOUTIT_ENV` should be one of these:
    - `local`
    - `dev`
    - `prod`
- `SHOUTIT_DEBUG` can be `True`
- `RAVEN_DSN`

- `SITE_LINK`, `API_LINK`

- `DB_HOST`, `DB_PORT`, `DB_USER`, `DB_PASSWORD`
- `REDIS_HOST`, `REDIS_PORT`
- `ES_HOST`, `ES_PORT`

- `EMAIL_USING`
- `PUSHER_ENV`
- `TWILIO_ENV`
- `MIXPANEL_TOKEN`
