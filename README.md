- `API_LINK`
- `DB_HOST`, `DB_PORT`, `DB_USER`, `DB_PASSWORD`
- `EMAIL_ENV`
- `ES_HOST`, `ES_PORT`, `ES_BASE_INDEX`
- `FACEBOOK_APP_ID`, `FACEBOOK_APP_SECRET`
- `FORCE_SYNC_RQ`
- `MIXPANEL_TOKEN`
- `PUSHER_ENV`
- `RAVEN_DSN`
- `REDIS_HOST`, `REDIS_PORT`
- `SHOUTIT_DEBUG` any true value
- `SHOUTIT_ENV` should be one of these:
    - `local`
    - `dev`
    - `prod`
- `SITE_LINK`
- `TWILIO_ENV`
